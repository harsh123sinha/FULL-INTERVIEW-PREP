Steps to Create a Resume on Overleaf

  • Visit Overleaf's website.
  • Sign Up or Log In: If you're new to Overleaf, sign up; otherwise, log in.
  • Start a New Project:
    • On the top left corner, find the "New Project" button and click it.
    • Choose a project name for your resume and proceed.
  • Code Editor Setup:
    • Overleaf will open a split-screen editor with a code editor on the left and a preview on the right.
    • In the left panel (code editor), paste the provided LaTeX code below.
  • Compile the Document:
    • Click the "Recompile" button on the left to render your resume preview.
    • Make adjustments as needed directly in the code editor.
  • Download Your Resume:
    • When youâ€™re satisfied with the resume, go to the right corner of the screen.
    • Click the "Download" button to save your finished resume.
